# P01: GitHub. Treballant de forma col·laborativa: forks i pull request.

## Descripció del Projecte
Aquesta activitat té com a objectiu introduir els estudiants en el flux de treball col·laboratiu de Git i GitHub utilitzat en entorns professionals reals com la consultora EverPia. Deixant de banda el treball estrictament individual o l'ús d'una única branca centralitzada ("main"), es simularà el procés d'incorporació (*onboarding*) de nous consultors mitjançant un sistema jeràrquic i un procés estricte de revisió de codi basat en *Forks* i *Pull Requests*.

El domini d'aquestes competències és fonamental per garantir la coordinació, organització i traçabilitat del codi o fitxers de configuració dins del sector IT.

---

## Flux de Treball a EverPia
L'activitat es desenvoluparà en els grups de treball habituals, on s'hauran de definir clarament els següents rols:
* **Team Leader (Líder d'Equip):** Encarregat de coordinar el repositori i supervisar el procés d'integració.
* **Membres de l'Equip:** Consultors encarregats de realitzar les bifurcacions (*forks*), treballar en local i sol·licitar la fusió de canvis mitjançant *Pull Requests*.
