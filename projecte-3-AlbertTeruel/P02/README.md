# P02: Repositori de GitHub

### Breu descripció
Ara ja sabem que **GitHub** ens proporciona una plataforma molt potent per gestionar la nostra documentació de projectes. Per això en aquest tercer projecte, al repositori que haureu creat a la tasca 0, caldrà que tingueu una carpeta per cadascuna de les activitats que tinguin lliurament i que allà tingueu les guies, documentacions, configuracions, etc. que es demanin.

---

Per una correcta gestió és molt important que es mantingui un **criteri uniforme**:

* **Arxiu arrel:** A l’arxiu `README.md` de presentació del projecte s'han d'incloure enllaços directes a les diferents carpetes.
* **Organització de tasques:** Crear carpetes específiques per a les diferents tasques (`tasca01`, `tasca02`, etc.) on tindreu un arxiu `README.md` amb l’enunciat o descripció i l’enllaç a l’arxiu que contingui la solució de l’activitat.
* **Organització de productes:** En el cas dels productes que tinguin lliurament, creareu una carpeta que s’anomeni `producte04`, per exemple, i se seguirà exactament el mateix criteri que pel lliurament de les tasques.
