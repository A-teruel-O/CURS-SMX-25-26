# P03: Rèplica web

## Descripció del Projecte
Després de superar la primera fase de supervivència tècnica a EverPia, l’equip de la consultora ha rebut un nou encàrrec d’un client exigent: replicar en local una web corporativa real feta amb WordPress. L'objectiu és analitzar-ne l’estructura, entendre’n l’arquitectura de continguts i aprendre bones pràctiques de desenvolupament i manteniment web professional.

Aquest repte s’emmarca dins l’escenari de “Sobreviure en una empresa IT”, on s'ha de treballar amb criteri, rigor tècnic i capacitat d’adaptació a webs complexes. En aquest cas, es realitzarà una còpia de la web de referència `davidperalvarez.com`, un lloc que combina claredat, estructura SEO, disseny net i continguts professionals.

**Objectiu final:** Aprendre a replicar fidelment una web existent, tant a nivell de pàgines i enllaços com d’aparença i organització interna, treballant des d’un entorn local de proves amb WP Local.

---

## Instruccions de Realització

### 1. Entorn de treball
* Utilitzeu l'eina **WP Local** instal·lada al PC de classe.
* Creeu una nova web seguint la nomenclatura: `replica_[nom]` (per exemple: `replica_Cristian`).

### 2. Investigació prèvia
Abans de començar la construcció, cal explorar i analitzar la web original:
* **Navegació principal:** Estudiar l'organització de menús i submenús.
* **Estructura de continguts:** Analitzar la distribució de textos, imatges, vídeos i seccions.
* **Estil visual:** Identificar colors, tipografies i composició general.
* **Eines recomanades per a l'anàlisi:**
  * Detector de temes i plugins: `wpthemedetector.com`
  * Extractor de colors exactes: `imagecolorpicker.com/ca`
  * Identificador de fonts tipogràfiques: `myfonts.com/es`
  * Extracció de la llista completa d'URLs del lloc mitjançant el sitemap de pàgines: `davidperalvarez.com/page-sitemap.xml`
  * Plugin de blocs recomanat per a la maquetació: **Kadence Blocks – Gutenberg Blocks for Page Builder Features**.

### 3. Creació de pàgines
Cada consultor ha de construir manualment de forma individual totes les pàgines que apareixen al sitemap del web original, incloent-hi com a mínim:
* Inici
* Sobre mí
* Academia básica
* Mantenimiento para academias online
* Consultoría para academias online
* Proyectos que he realizado
* Descarga
* Contactar
* Pàgines legals: Política de privacidad, cookies, condiciones de uso, etc.

*Nota: Els continguts es poden reproduir amb el text copiat directament de la web o generant text de farcit (Lorem Ipsum), sempre que es mantinguin els títols, les seccions i l'estructura visual originals.*

### 4. Configuració del lloc
* Definir una pàgina d’inici estàtica.
* Configurar el menú principal i els submenús de navegació de manera idèntica a l'original.
* Activar i provar diferents temes de WordPress fins a trobar-ne un que s’ajusti correctament al disseny.
* Ajustar els enllaços permanents (*permalinks*) i els paràmetres generals del lloc (idioma, zona horària, nom del lloc, etc.).
