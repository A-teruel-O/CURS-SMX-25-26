# P01: Kanban & Compte de Github

## Descripció del Projecte
Aquest projecte integra dues eines estructurals per al flux de treball professional dins la consultora EverPia: la gestió visual de projectes mitjançant la metodologia Kanban a Microsoft Planner i la configuració d'un entorn de control de versions i documentació tècnica individual a GitHub. El propòsit és establir un sistema d'organització transparent, traçable i alineat amb els estàndards del sector IT.

---

## Objectius Específics
* Establir una eina de coordinació col·laborativa i centralitzada per a cada grup de treball.
* Garantir un espai individual de desenvolupament, control de versions i documentació tècnica per a cada alumne.
* Aplicar de forma pràctica els principis de transparència i organització de la consultora EverPia.

---

## Estructura de la Tasca

### Part 1: Tauler Kanban (Microsoft Planner) - Àmbit Grupal
Cada grup de treball ha de crear i administrar un tauler Kanban a Microsoft Planner subjecte a les següents especificacions de flux:

* **Columnes mínimes obligatòries:**
  * `Backlog`: Llistat centralitzat de totes les tasques pendents del projecte.
  * `En curs`: Tasques en fase activa de desenvolupament.
  * `En revisió`: Tasques finalitzades pendents de validació i control de qualitat.
  * `Fet`: Tasques completades definitivament.
* **Gestió del Backlog:** S'ha d'inicialitzar el tauler des de l'inici del projecte incloent totes les tasques planificades (des de T01 fins a T10). La progressió i desplaçament de les targetes entre columnes s'ha d'actualitzar en temps real segons l'estat del projecte.

### Part 2: Compte i Repositori GitHub - Àmbit Individual
Cada membre del grup ha de configurar de manera individual el seu entorn de codi:

1. **Creació de compte:** Registre o ús d'un compte personal a GitHub associat obligatòriament a l'adreça de correu electrònic institucional de l'escola.
2. **Repositori de perfil:** Creació del repositori de presentació personal (repositori especial amb el mateix nom d'usuari).
3. **Documentació:** Redacció i estructuració del fitxer `README.md` inicial amb la informació de la presentació personal de l'estudiant.

---

## Requisits dels Entregables
Per a l'avaluació de l'activitat s'han de lliurar els següents accessos directes:
* **Entregable Grupal:** Enllaç públic o configurat amb permisos compartits per a l'accés docent al tauler Kanban de Microsoft Planner.
* **Entregable Individual:** Enllaç directe al repositori personal de GitHub de l'alumne.
