# T11: Instal·lació de WordPress en local amb WP Local – Aplicacions Web

### Breu descripció
En aquesta tasca, cada alumne crearà la seva primera instal·lació de **WordPress** en local utilitzant el programari **WP Local**, el qual ja es troba instal·lat als ordinadors de l’aula.

Aquest WordPress actuarà com el vostre entorn de proves operacional. Es farà servir durant les classes magistrals per comprendre els fonaments de la plataforma i preparar el desplegament d’una rèplica d’una web completa en el pròxim projecte.

Com que la finalitat d'aquesta web és purament d'aprenentatge, els continguts estructurals no són crítics en aquesta fase. S’haurà d’utilitzar text i imatges de farcit (*dummies*):

* **Text de farcit (*Dummy text*):** Utilitzeu la plataforma [Lipsum](https://www.lipsum.com/) per generar paràgrafs de text sense sentit per omplir els espais.
* **Imatges de farcit (*Dummy images*):** S'estableix l'ús de [Picsum Photos](https://picsum.photos).
  * Per a imatges quadrades: especifiqueu una única dimensió en píxels (exemple: `https://picsum.photos/200`).
  * Per a dimensions concretes: afegiu ambdues dimensions separades per barra (exemple: `https://picsum.photos/200/300`).

El professorat realitzarà una classe magistral explicativa i, posteriorment, procedireu de forma autònoma a la instanciació i parametrització de la vostra pàgina de proves.
